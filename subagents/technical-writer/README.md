# Technical Writer Subagent

## Overview
The Technical Writer Subagent is a specialized AI component designed to generate high-quality technical documentation for Physical AI and Humanoid Robotics projects.

## Features
- Generates Docusaurus-compatible MDX content
- Creates production-ready code examples with comments
- Includes Mermaid diagrams for visual explanations
- Follows consistent educational structure
- Maintains technical accuracy across domains

## Usage
```bash
claude-code subagent:technical-writer --topic "Your topic here"
```
