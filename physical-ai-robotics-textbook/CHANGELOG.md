# Changelog

All notable changes to the Physical AI & Humanoid Robotics Textbook project will be documented in this file.

## [Unreleased]

## [1.2.0] - 2025-12-31

### Added
- **Multi-Language Translation System**
  - Added Urdu translation with RTL layout support
  - Added Hindi translation with LTR layout support
  - Added comprehensive translation infrastructure
  - Added TranslateButton component for chapter-specific language switching
  - Added language preference persistence using localStorage and cookies
  - Added automatic language detection from browser settings
  - Added fallback mechanism for missing translations
  - Added user profile integration for language preferences
  - Added Noto Nastaliq Urdu and Noto Sans Devanagari fonts
  - Added comprehensive RTL CSS support for Urdu
  - Added language switching API endpoints
  - Added user guide for language switching features
  - Added developer guide for adding new languages
  - Added translation contribution guidelines
  - Added i18n utilities documentation
  - Added comprehensive README updates with translation information

### Changed
- Updated README with comprehensive translation system documentation
- Enhanced authentication system to support language preferences
- Improved documentation structure to support multiple languages
- Updated build system to support multi-locale builds
- Enhanced CSS to support RTL layouts and multiple fonts
- Updated navigation to include language switching dropdown

### Fixed
- Various styling issues for RTL layouts
- Font loading and rendering issues for non-Latin scripts
- Accessibility issues in multilingual context
- Language preference persistence across sessions

### Security
- Enhanced security for translation API endpoints
- Improved validation for locale switching
