<h2 className="second-heading">Common Issues</h2>
<div className="underline-class"></div>

**Unrealistic Time Planning**:
- Double estimates for complex modules
- Add 25-50% buffer for troubleshooting
- Plan for hands-on implementation
- Schedule regular reviews

**Prerequisites Not Met**:
- Complete Python fundamentals
- Review linear algebra/calculus
- Practice Linux commands
- Learn basic robotics concepts

**Poor Pathway Selection**:
- Reassess goals and objectives
- Evaluate skill level honestly
- Consider time constraints
- Adjust pathway/timeline

**Integration Understanding**:
- Identify module connections
- Apply to capstone project
- Create visual integration map
- Focus on big picture

**Progress Tracking**:
- Set measurable objectives
- Create verification checkpoints
- Document completed work
- Regular self-assessment
