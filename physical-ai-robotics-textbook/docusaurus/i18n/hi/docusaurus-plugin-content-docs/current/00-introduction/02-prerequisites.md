<h2 className="second-heading">सामान्य समस्याएं</h2>
<div className="underline-class"></div>

**आरओएस 2 स्थापना समस्याएं**:
```bash
lsb_release -sc  # उबंटू संस्करण की पुष्टि करें
sudo apt update
curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
ros2 --version  # स्थापना की पुष्टि करें
```

**अनुमति समस्याएं**:
```bash
sudo usermod -a -G dialout $USER
sudo usermod -a -G plugdev $USER
newgrp dialout
groups $USER  # समूहों की पुष्टि करें
```

**पायथन पैकेज समस्याएं**:
```bash
source /opt/ros/humble/setup.bash
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc
python3 -c "import rclpy; print('ROS Python libraries available')"
```

**कोलकॉन बिल्ड निर्भरताएं**:
```bash
cd ~/ros2_ws
rosdep install --from-paths src --ignore-src -r -y
colcon build
```

**नेटवर्क संचार समस्याएं**:
```bash
echo $ROS_DOMAIN_ID
export ROS_DOMAIN_ID=0
sudo ufw allow from [OTHER_IP] to any port 7400:7500 proto udp
ros2 topic list
```
