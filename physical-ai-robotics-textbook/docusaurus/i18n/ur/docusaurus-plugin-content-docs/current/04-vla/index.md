# <h1 className="main-heading">VLA: وائس ٹو ایکشن سسٹمز</h1>
<div className="underline-class"></div>

یہ ماڈیول وائس ٹو ایکشن (VLA) سسٹمز کو تلاش کرتا ہے جو ہیومنوائڈ روبوٹس کو انسانی وائس کمانڈز کو سمجھنے اور اس کا جواب دینے کے قابل بناتے ہیں۔ آپ سیکھیں گے کہ بڑے لینگویج ماڈلز (LLMs) اور اسپیچ ریکگنیشن کو کیسے یکجا کریں تاکہ قدرتی ہیومن-روبوٹ انٹرایکشن تخلیق کیا جا سکے۔

<h2 className="second-heading">
سیکھنے کے اہداف
</h2>
<div className="underline-class"></div>

اس ماڈیول کے اختتام تک، آپ کے استطاع میں ہو گا:
- • ہیومنوائڈ روبوٹس کے لیے اسپیچ ریکگنیشن سسٹمز نافذ کرنا
- • کمانڈ تشریح کے لیے بڑے لینگویج ماڈلز کو یکجا کرنا
- • وائس کمانڈز کے مطابق ایکشن پلاننگ سسٹمز تخلیق کرنا
- • روبوٹ کنٹرول کے لیے قدرتی لینگویج انٹرفیسز ڈیزائن کرنا
- • ملٹی موڈل ان پٹ پروسیسنگ (وائس + وژن) نافذ کرنا

<h2 className="second-heading">
ضروریات
</h2>
<div className="underline-class"></div>

اس ماڈیول کا آغاز کرنے سے پہلے، آپ کے پاس ہونا چاہیے:
- • پچھلے ماڈیولز (تعارف، ROS 2، سیمولیشن، Isaac) مکمل ہوں
- • قدرتی لینگویج پروسیسنگ کے تصورات کی سمجھ
- • API یکجہتی کا تجربہ
- • مشین لرننگ فریم ورکس کا بنیادی علم

<h2 className="second-heading">
ماڈیول جائزہ
</h2>
<div className="underline-class"></div>

یہ ماڈیول یہ سب کچھ شامل کرتا ہے:
1. وائس ٹو ایکشن سسٹمز کا تعارف
2. اسپیچ ریکگنیشن اور پروسیسنگ
3. بڑے لینگویج ماڈل یکجہتی
4. کمانڈ تشریح اور پلاننگ
5. ملٹی موڈل انٹرایکشن سسٹمز
6. وائس انٹرفیس ڈیزائن پیٹرنز
